# About
I’m a thesis-based Master’s student at McGill University graduating **May 2026**, focused on building scalable and reliable data-driven systems.

I spent three years as a Big Data Developer in American Express’s AML domain, building batch ETL pipelines and automated reporting workflows using **SQL, Hive, Spark, Python, and Unix shell**. I owned two major reports end to end and built ingestion table pipelines supporting downstream analytics.

In my thesis, I evaluate LLM-based pattern discovery and build scalable validation tooling using a domain-specific language and automated Java validator generation, and I also implement statistical pattern mining using FP-Growth to extract high-confidence patterns.
